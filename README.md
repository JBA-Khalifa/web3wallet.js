# web3wallet.js

Wrap `@polkadot-js/api` to provide APIs for web3-setters-club/web3wallet-sdk.

@polkadot-js/api: ^v1.29.1

## build & test

To build:

```bash
yarn install
yarn run build
```

To test:
open `./test/index.html` in chrome.
open chrome dev console and run `runTests().then(console.log)`
